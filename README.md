## Installation
1. Clone the repository.

    ```
    git clone https://github.com/ameybhavsar24/githubfinder.git
    ```

2. Change directory to `githubfinder/`

    ```
    cd githubfinder/
    ```

3. Install the dependencies.

    For `npm`:
    ```
    npm i
    ```

    For `yarn`:
    ```
    yarn install
    ```
